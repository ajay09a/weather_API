🌦️ Weather App (Next.js)

A simple weather application built using Next.js that allows users to search for any city and view weather information for:

📍 Today (current weather)
📅 Yesterday
📅 Tomorrow


🚀 Features
🔍 Search weather by city name
⌨️ Press Enter or click button to search
🌡️ Displays temperature, wind speed, and conditions
📊 Shows 3-day data (yesterday, today, tomorrow)
⚡ Fast and responsive UI
❌ Error handling for invalid input


🛠️ Tech Stack
Next.js (App Router)
React (useState)
CSS (custom styling)
Visual Crossing Weather API
